# BadApplePFP

Create bad apple renderings out of your social profile images.

## Usage

You will need:

- git
- linux
- curl
- ffmpeg
- golang

Steps:

```bash
git clone https://github.com/vcokltfre/badapplepfp.git
curl -o profile.png <your profile url>
sh build.sh
```

Your produced video will be in `final.mp4`
